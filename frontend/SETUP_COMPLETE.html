Quick Start (5 Minutes)

  1. 1 Go to MongoDB Atlas and click "Try Free"
  2. 2 Sign up → Create Project → Create Free Cluster (2-5 min wait)
  3. 3 Create database user (username: organicfruits) → Copy connection string
  4. 4 Paste connection string in: backend-java/src/main/resources/application.properties
  5. 5 Restart backend and you're done! ✅
