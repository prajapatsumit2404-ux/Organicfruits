🚀 MongoDB Atlas Setup

Follow all steps to connect your cloud database

1

Create MongoDB Atlas Account

Visit https://www.mongodb.com/cloud/atlas and sign up

✓ Click "Try Free"
✓ Sign up with email or Google
✓ Verify your email

2

Create a Project

Set up your first project

✓ Click "Create a Project"
✓ Name: OrganicFruits
✓ Click "Create Project"

3

Create a Cluster

Create a free database cluster

✓ Click "Create"
✓ Choose "Free" tier (M0)
✓ Pick your region
✓ Name: OrganicFruits-Cluster
✓ Click "Create"
⏳ Wait 2-5 minutes

4

Create Database User

Set up authentication credentials

✓ Click "Database Access" (left menu)
✓ Click "Add New Database User"
✓ Username: organicfruits
✓ Password: Create strong password (save it!)
✓ Privileges: readWriteAnyDatabase
✓ Click "Add User"

5

Allow Network Access

Configure IP whitelist

✓ Click "Network Access" (left menu)
✓ Click "Add IP Address"
✓ Select "Allow Access from Anywhere"
✓ (Or enter your IP for security)
✓ Click "Confirm"

6

Get Connection String

Copy your MongoDB connection string

✓ Go to "Clusters"
✓ Click "Connect"
✓ Select "Drivers"
✓ Choose Node.js
✓ Copy the connection string

Connection String (save this):

7

Update Backend Configuration

Configure your backend to use MongoDB Atlas

⚠️ IMPORTANT:

1. Open: backend\.env

2. Find: MONGODB_URI=...

3. Replace with your connection string from Step 6

4. Replace PASSWORD with your actual password

5. Save the file

Example:
MONGODB_URI=mongodb+srv://organicfruits:PASSWORD@cluster.xxxxx.mongodb.net/organicfruits?retryWrites=true&w=majority

8

Restart Backend

Start your Node.js server

cd e:\hotwax\backend
npm start

Look for this message:

✅ Connected to MongoDB

✅ All Set!

Your MongoDB Atlas is now connected!

✓ Frontend: http://localhost:8000

✓ Backend: http://localhost:5000

✓ Database: MongoDB Atlas (Cloud)

Next: Open http://localhost:8000 and register your first account!
